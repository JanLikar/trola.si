Simple webapp for Ljubljana bus traffic info (LPP).

.. warning:: it's not possible to open source this since it contains secret keys (sentry ,..)

INSTALL
=======

::
    $ python bootstrap.py
    $ bin/buildout
    $ bin/python trolasi/__init__.py
